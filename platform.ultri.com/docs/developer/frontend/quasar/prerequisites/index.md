# Ultri Quasar / Vue3 site app prerequisites

* Yarn
* Quasar CLI


## Yarn

[Installation - Ubuntu](https://classic.yarnpkg.com/lang/en/docs/install/#debian-stable)

## Quasar CLI

Install Quasar CLI a a global npm.

```sh
yarn global add @quasar/cli
```
